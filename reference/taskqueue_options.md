# Set or get options for my package

Set or get options for my package

## Usage

``` r
taskqueue_options(...)
```

## Arguments

- ...:

  Option names to retrieve option values or `[key]=[value]` pairs to set
  options.

## Value

Options for task queue

## Supported options

- `host`: Host of PostgreSQL database

- `port`: Port of PostgreSQL database

- `user`: User name of PostgreSQL database

- `password`: Password of PostgreSQL database

- `database`: Database name of PostgreSQL database
